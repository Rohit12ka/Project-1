# Task Manager

## Project Description
Task Manager is a simple and interactive web application developed using HTML, CSS, and JavaScript. It allows users to efficiently manage their daily tasks by adding new tasks, marking tasks as completed, deleting tasks, and tracking task statistics in real time.

### Features
* Add new tasks
* Mark tasks as completed/incomplete
* Delete tasks
* Live task statistics:
                      * Total Tasks
                      * Completed Tasks
                      * Pending Tasks
                      * Responsive and user-friendly interface


## Technologies Used
* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## Steps to Run the Project

### Method A: Open Directly
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.

### Method B: Using VS Code
1. Open the project folder in VS Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.
The application will open in your browser.

## How to Use
1. Enter a task in the input field.
2. Click the **Add** button or press **Enter**.
3. Click the checkbox to mark a task as completed.
4. Click the checkbox again to mark it as incomplete.
5. Click the **Delete** button to remove a task.
6. View live statistics for Total, Completed, and Pending tasks.
